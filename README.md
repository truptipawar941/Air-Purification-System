# Smart Air Purification System

## Project Overview
Smart Air Purification System is an IoT-based project designed to monitor air quality and maintain a healthier environment. The system uses sensors to detect air quality parameters and provides real-time monitoring with automated control.

## Features
- Real-time air quality monitoring
- Detection of harmful gases and environmental conditions
- Automated purification control
- Sensor-based data collection
- IoT-enabled smart monitoring

## Components Used
- ESP8266 / ESP32 Microcontroller
- MQ-135 Gas Sensor
- DHT22 Temperature and Humidity Sensor
- Relay Module
- Fan/Purification Unit
- Jumper Wires
- Power Supply

## Technologies Used
- Arduino IDE
- Embedded C
- IoT
- ESP8266/ESP32
- Sensors

## Working
The MQ-135 sensor detects air quality levels, while the DHT22 sensor measures temperature and humidity. The microcontroller processes sensor data and controls the purification system automatically based on the detected air quality conditions.

## Applications
- Smart homes
- Indoor air quality monitoring
- Healthcare environments
- Offices and classrooms

## Future Scope
- Mobile application integration
- Cloud-based data storage
- AI-based air quality prediction
- Advanced purification control
